### array

Un array se puede declarar usando el tipo de dato que contendrá el array:

```ts
let names: string[] = [];
names.push('Jon');
names.push('Ane');
```

O bien usando el tipo genérico Array, especificando el tipo de sus elementos:

```ts
let names: Array<string> = [];
names.push('Jon');
names.push('Ane');
```
