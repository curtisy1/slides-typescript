## Parámetros por defecto


Se pueden declarar los parámetros por defecto:

```ts
function saluda(aQuien = 'mundo'): void {
  console.log(`Hola ${aQuien}`);
}

saluda("Pepito"); // Hola pepito
saluda(); // Hola mundo
saluda(undefined); // Hola mundo
```
